# مرحب بيك في عالمي الإبداعي الرقمي! 🎨👋

أنا **حامد**، طالب في كلية الصيدلة ومصمم جرافيك شغوف ببناء الهويات البصرية ودمج الفن والتصميم مع العلوم الطبية والتكنولوجيا.

## 🩺 الصيدلة والتكنولوجيا الطبية
- طالب صيدلة مهتم بالابتكار الرقمي وتوظيف التقنية لخدمة القطاع الصحي والصيدلاني.

## 🎨 التصميم والجرافيك (مجالي الأساسي)
- **بناء الهويات البصرية:** تصميم الشعارات (Logos) والبراندنج الكامل والخطوط المبتكرة للمشاريع والمؤسسات.
- **تطوير الهوية البصرية:** دمج الرموز التاريخية والثقافية بأسلوب عصري لخدمة البراندات الناشئة.
- **الأدوات:** متمكن من أدوات التصميم السريعة والذكية مثل Canva و PixelLab.

## 💻 البرمجة والتقنية
- في رحلة تعلم مستمرة لأساسيات البرمجة لربط الرؤية الفنية الإبداعية بالتنفيذ البرمجي في الصيدلة الرقمية.
